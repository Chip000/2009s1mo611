\section{Conclusão}
\label{sec:conc}

O \wa{} é um problema chave para o gerenciamento das redes WDM.
Este problema está na classe NP-difícil, mas utilizando a Teoria da
Complexidade Parametrizada foi possível obter um algoritmo exato que 
pode explorar características não triviáis do \wa{}. A Teoria de Complexidade
Parametrizada define novos horizontes para a aplicação na alocação de comprimentos
de ondas em redes transparentes\footnote{Para outros problemas também.}, abrindo
espaço para muitos outros trabalhos que utilizam essa técnica.
